\# Agent Instructions (Solo Project)



This repository is maintained by a single developer.



\## General Rules

\- Do not create new files unless explicitly requested.

\- Do not modify files other than those explicitly named.

\- Do not add documentation, guides, or meta files.

\- Do not create branches or pull requests.

\- Work directly on the current branch.



\## Editing Rules

\- When asked to modify a file, make the minimum necessary changes.

\- Preserve existing structure and behavior unless explicitly told otherwise.

\- Do not refactor unrelated code.

\- Do not rename variables, functions, or UI elements unless requested.



\## Output Rules

\- After making any changes, output full contents of index.html and any other changed files in .md format.

\- Do not output diffs unless explicitly requested.

\- Do not use markdown code fences unless explicitly requested.



\## Git Rules

\- Only commit when explicitly instructed.

\- Only commit files that were explicitly modified.

\- Do not invent reasons to commit.



\## UI / Audio Project Specifics

\- This project prioritizes stability over novelty.

\- Avoid cascading UI changes.

\- Audio playback must not regress when adding features.
